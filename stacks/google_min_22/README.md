A minimal buildpack stack based on Ubuntu 22.04 LTS (Jammy Jellyfish).

## Run Image

[gcr.io/buildpacks/stacks/google-min-22/run](https://gcr.io/buildpacks/stacks/google-min-22/run)

* `ca-certificates`
* `locales`
* `openssl`
* `tzdata`
